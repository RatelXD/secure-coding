## B.2 결과 해석

현재 자동 테스트는 아이디 정규화, 비밀번호 단방향 해시 저장, 상품 소유자 관계, 채팅 입력·재전송 정책, 신고 임계값 정책, 프록시 헤더 신뢰 경계를 확인합니다. 실제 HTTP·WebSocket 진입점과 브라우저 화면이 생기면 같은 정책이 모든 진입점에 적용되는지 별도의 통합·종단 테스트가 필요합니다.
